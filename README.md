# dvd-woo-pagseguro-parcelas
Plugin do wordpress para o woocommerce para mostrar parcelamento do pagseguro 

## Funcinalidades

1. Mostrar Parcelamento
2. Desconto no Boleto
3. Tabela de Parcelamento na pagina do produto


link plugin
